# Intro

Proyecto utilizando Django para el backend y React en el frontend.

Deployment utilizando Docker y subido a Heroku: https://heyheymydjreact.herokuapp.com/

 
    
## Technologies
    
    Python 3.7
    
    Django==3.0.8
    django-cors-headers==3.4.0
    djangorestframework==3.11.0
    gunicorn==20.0.4
    psycopg2-binary
    
    react: 16.13.1
    redux: ^4.0.5
    axios: 0.19.2
    antd: 4.4.0
